\section{Introduction}

\begin{primarybox}[Overview]
\textbf{TruthGaurd} is a comprehensive fraud detection mobile application designed to protect users from deepfake media and fraudulent job postings. The app leverages artificial intelligence and machine learning to provide real-time threat detection and analysis, making digital fraud prevention accessible to everyone.

In today's digital age, deepfake technology has evolved from a technical curiosity to a significant threat to digital trust and personal security. Simultaneously, online job platforms have become breeding grounds for sophisticated employment scams. TruthGaurd addresses these critical needs by combining advanced machine learning models with an intuitive mobile interface, enabling users to verify the authenticity of digital content and job postings before interaction.

The application integrates multiple security features including image/video deepfake detection, job posting fraud analysis, an AI-powered security chatbot, and a community platform for sharing threat intelligence. By providing instant threat assessment with confidence scores and detailed explanations, TruthGaurd empowers users to make informed decisions about their digital interactions.

Built with modern technologies including React Native for cross-platform compatibility, Flask-based ML services for intelligent threat detection, and Supabase for robust authentication and data management, TruthGaurd represents a holistic approach to mobile fraud prevention that prioritizes both security and user experience.
\end{primarybox}

\section{App Purpose}

TruthGaurd addresses the growing concern of digital fraud and online deception in today's interconnected world. With the increasing sophistication of deepfake technology and job posting scams, users need a reliable tool to verify the authenticity of digital content before interaction. The application serves to:

\begin{itemize}
    \item Detect and analyze deepfake images and videos
    \item Identify fraudulent job postings to protect job seekers from employment scams
    \item Provide real-time fraud detection using AI/ML models
    \item Educate users about fraud threats through an AI assistant powered by Google Gemini Pro
    \item Create a community platform for sharing security insights and experiences
    \item Track and monitor detection activity for user awareness
\end{itemize}

The app's AI-powered detection system provides instant analysis of potential threats, helping users make informed decisions about their digital interactions and protecting them from financial and emotional harm.

\section{Target Audience}

TruthGaurd targets a diverse audience of individuals vulnerable to digital fraud, including:

\begin{itemize}
    \item Job seekers who encounter suspicious job postings on online platforms
    \item Social media users exposed to deepfake videos and manipulated media
    \item Journalists and fact-checkers who need to verify media authenticity
    \item Elderly users vulnerable to sophisticated digital scams
    \item Students and young professionals entering the job market
    \item Business professionals who conduct important communications through digital channels
    \item Online shoppers who encounter suspicious content
\end{itemize}

The app is particularly valuable for users who frequently consume social media content, participate in online job markets, or need to verify the authenticity of digital media.

\section{Core Features}

\subsection{AI-Powered Fraud Detection}

The core functionality of TruthGaurd revolves around intelligent fraud detection using machine learning models. These features enable users to analyze and verify potentially fraudulent content before interaction.

\subsubsection{Deepfake Detection}
\begin{primarybox}[Deepfake Detection]
The Deepfake Detection system analyzes images and videos for signs of AI manipulation and synthetic content generation.
\begin{itemize}
    \item \textbf{Image analysis:} Upload PNG, JPG, JPEG images for deepfake detection
    \item \textbf{Video analysis:} Supports MP4, AVI, MOV video formats for frame-by-frame analysis
    \item \textbf{CNN-based detection:} Pretrained convolutional neural network identifies manipulation patterns
    \item \textbf{Multi-factor analysis:} Blur detection, artifact identification, and consistency checking
    \item \textbf{Risk scoring:} 0-100 scale indicating likelihood of deepfake content
    \item \textbf{Real-time processing:} Results returned within 30-40 seconds depending on file size
\end{itemize}
\end{primarybox}

\subsubsection{Job Fraud Detection}
\begin{primarybox}[Job Fraud Detection]
The Job Fraud Detector analyzes job postings for scam indicators and fraudulent patterns using natural language processing.
\begin{itemize}
    \item \textbf{Text analysis:} Examines job title, description, requirements, and salary information
    \item \textbf{Keyword detection:} Identifies 50+ known fraud indicators and suspicious phrases
    \item \textbf{Pattern recognition:} Detects unrealistic salary claims and urgency tactics
    \item \textbf{Legitimacy validation:} Recognizes authentic employment signals like benefits and career growth
    \item \textbf{Company verification:} Validates company name presence and credibility
    \item \textbf{Risk assessment:} Calculates fraud probability with specific red flags
    \item \textbf{Instant results:} Analysis completed in under 30 seconds
\end{itemize}
\end{primarybox}

\section{System Architecture}

\begin{primarybox}[Architecture Overview]
TruthGaurd follows a modern three-tier microservices architecture:

\vspace{0.5cm}
\textbf{1. Mobile Frontend (React Native + Expo)}
\begin{itemize}
    \item Cross-platform iOS and Android support
    \item TypeScript for type safety and compile-time error prevention
    \item Expo Router for file-based navigation
    \item Context API for global state management (AuthContext)
    \item AsyncStorage for persistent local data
    \item Custom themed components with gradient design
\end{itemize}

\vspace{0.5cm}
\textbf{2. Backend API Gateway (Node.js + Express)}
\begin{itemize}
    \item RESTful API architecture with JSON responses
    \item Multer middleware for multipart file uploads
    \item Axios HTTP client for ML service communication
    \item CORS-enabled for cross-origin requests
\end{itemize}

\vspace{0.5cm}
\textbf{3. ML Service (Python + Flask)}
\begin{itemize}
    \item Separate microservice for ML model inference
    \item Flask REST API for model serving
    \item OpenCV image/video processing
    \item NLP-based job fraud classification
\end{itemize}

\vspace{0.5cm}
\textbf{4. External Services}
\begin{itemize}
    \item Supabase for authentication and database
    \item Google Gemini Pro for AI chatbot conversations
\end{itemize}
\end{primarybox}


\section{Security Implementation}

\begin{primarybox}[Security Measures]
\textbf{Authentication \& Authorization}
\begin{itemize}
    \item \textbf{Supabase Authentication:} Enterprise-grade auth with encrypted password storage
    \item \textbf{JWT tokens:} Stateless authentication for secure session management
    \item \textbf{Secure token storage:} AsyncStorage encrypts and protects tokens on device
    \item \textbf{Automatic token refresh:} Maintains persistent sessions without re-login
    \item \textbf{Password recovery:} Secure reset flow via email verification
    \item \textbf{Role-based access:} Admin privileges for content moderation
\end{itemize}

\vspace{0.5cm}
\textbf{Data Protection}
\begin{itemize}
    \item \textbf{HTTPS communications:} All API requests encrypted with SSL/TLS
    \item \textbf{Environment variables:} Sensitive credentials isolated from source code
    \item \textbf{No media retention:} Uploaded files deleted immediately after analysis
    \item \textbf{Automatic cleanup:} Temporary files removed from backend and ML service
    \item \textbf{Input validation:} File type and size restrictions prevent malicious uploads
    \item \textbf{Supabase RLS:} Row-level security policies protect user data
\end{itemize}

\vspace{0.5cm}
\textbf{API Security}
\begin{itemize}
    \item \textbf{CORS configuration:} Restricts API access to trusted origins
    \item \textbf{File size limits:} 50MB maximum prevents denial-of-service attacks
    \item \textbf{File type validation:} Whitelist approach allows only safe extensions
    \item \textbf{Error handling:} Generic error messages prevent information leakage
    \item \textbf{Request logging:} Audit trails for security monitoring
\end{itemize}

\end{primarybox}
